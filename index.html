Linux学习笔记--初次安装常用命令

常用命令

tty 打印当前连接终端 (Print the file name of the terminal connected to standard input.)
who 登录的用户(Print information about users who are currently logged in.)
whoami 当前用户
who am i 详细
sudo -i 切换城root用户

hostname -I 获取网络IP

-I, –all-ip-addresses
Display all network addresses of the host. This option enumerates all configured addresses on all network interfaces. The loopback interface and IPv6 link-local
addresses are omitted. Contrary to option -i, this option does not depend on name resolution. Do not make any assumptions about the order of the output.

Ctrl+Alt+F1和Ctrl+Alt+F2 字符界面和图形界面切换
free -h 查看内存占用
runlevel 当前的模式 5代表图形 3代表字符界面
init 3 切换字符界面
init 5 切换图形界面
init 0 关机
init 6 重启

cat /etc/motd 查看文本
nano 查看文本，可以编辑
/etc/motd 登录之后显示的文件
/etc/issue 登录之前显示的文件
echo 输出
ps aux查看进程
echo $SHELL 查看当前使用的shell
cat /etc/shells 查看当前系统支持的所有shell
hostname 查看主机名 在文件 /etc/hostname
hostname xxx 临时修改主机名 在内存中，重启失效
hostnamectl set-hostname XXX 内存和文件都改了，重启依然有效
echo PS1 提示格式(PS1前面加美元符号) 可以改，没必要
