01MySQL基础内容

登录详细规则
mysql -u root -pabc123 -hlocalhost -P3306

登录之后的操作
alter user ‘root‘@’localhost’ identified by ‘newpassword’; 修改root密码
show databases; 查看数据库
create database dbtest1; 创建数据库
use dbtest1; 选择使用某个数据库
create table employees(id int,name varchar(15)); //创建数据库
show create table employees; //显示建表语句
show create database dbtest1;//显示建数据库语句
show variables like ‘character_%’;
drop database dbtest1;删掉数据库

SQL分类

DDL（Data Definition Languages 、数据定义语言）这些语句定义了不同的数据库、表、视图、索引等数据库对象，还可以用来创建，删除，修改数据库和数据表的结构。

主要的语句关键字包括CREATE、DROP、ALTER等。

DML（Data Manipulation Language、数据操作语言）用于添加、删除、更新和查询数据库记录，并检查数据完整性。

  • 主要的语句关键字包括 INSERT、DELETE、UPDATE、SELECT等。
  • SELECT是SQL语言的基础，最为重要

DCL（Data Control Language、数据控制语言）用于定义数据库、表、字段、用户的访问权限的安全级别。

主要的语句关键字包括GRANT、REVOKE、COMMIT、ROLLBACK、SAVEPOINT等。

建议规范

数据库名，表明，字段名都用小写
关键字，函数名 用大写

注释

– 单行注释
/# 单行注释
/* */ 多行注释，不支持嵌套
