02.Linxu执行命令过程和命令类型

type 查看命令类型，是buildin还是外部命令

type -a 命令名 查看所有的

hash 查看缓存了的命令
hash -r 清空缓存

help和enbale 显示所有内部命令

  • enable cmd 启用内部命令

  • enable -n cmd 禁用内部命令

  • enable -n 查看所有禁用的内部命令

    以上临时禁用重启失效

which 查看外部命令的路径
whereis 除了查看外部命令的路径，还能显示帮助文档路径

. 配置文件 让系统重新读取配置文件
source 配置文件 让系统重新读取配置文件

alias cdnew=”cd /etc/sysconfig” 设置别名 临时设置
unalias cdnew 取消别名
unalias -a取消所有别名
alias 显示所有别名
不适用别名，执行原始命令

\命令
“命令”
‘命令’

别名放入文件
~/.bashrc 对自己有效才能持久保存
/etc/.bashrc 对所有用户生效

lsblk 显示硬盘和分区
